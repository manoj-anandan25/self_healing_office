# Runbook: Wi-Fi Outage

## Symptoms
- Users unable to connect to SSID
- Authentication failures (802.1X / RADIUS)
- High packet loss on wireless clients

## Steps
1. Verify AP status and controller logs.
2. Check SSID configuration and VLAN mapping.
3. Validate RADIUS server reachability.
4. Restart affected APs if isolated.
5. Escalate to network engineering if widespread.
